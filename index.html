Welcome to the Chit Picker Game

This festive game lets you register, log in, and pick chits to reveal random details! Each chit is a surprise, and the game is themed with Christmas and New Year cheer!

Game Rules to Play:

  • 1. Give Name, PHNo, Full Address while entering the details. Please give appropriate details [CaseSensitive 🔠].
  • 2. Login Using Name, PHNo [CaseSensitive 🔠].
  • 3. Only one person should play at once. Please check your Slack messages. You will receive a message with the text "Start Game" — only then should you play.
  • 4. You can only choose one chit, and then you should close the window [No Cheating 🙅‍♂️🙅‍♂️].
  • 5. Details should not be revealed to anyone [No Cheating 🙅‍♂️🙅‍♂️].
  • 6. Team Payments ➡️➡️➡️ GET,SET,GO 🔥🔥🔥
